Flow Layout

You actually know about flow layout, because it is what you have been using so far. It is the default layout algorithm. It arranges items in the inline (i.e. horizontal) direction and the block (i.e. vertical) direction. Generally items flow one after the other in one of these directions, generally without overlapping.
