Positioned Layout - Relative Positioning

Positioned Layout is different from Flow Layout in that it allows elements to take up the same space. Positioned layout uses the position property to move an element without affecting the position of other elements on the page. With relative positioning, the position of the element is shifted relative to its natural position.

For relative positioning, the css properties top, bottom, left, and right are used to shift an element as follows:

  • top:used to nudge the element down from its natural position, i.e. top: 10px shifts the element down by 10px.
  • bottom:used to nudge the element up from its natural position, i.e. bottom: 10px shifts the element up by 10px.
  • left:used to nudge the element to the right of its natural position
  • right:used to nudge the element to the left of its natural position
